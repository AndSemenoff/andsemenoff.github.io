Вопрос: Данные фрагменты кода выведут одно и тоже?
colors = ['red', 'blue', 'green', 'yellow']
result = ''
for s in colors:
    result += s
print(result)
colors = ['red', 'blue', 'green', 'yellow']
result = ''.join(colors)
print(result)
Выберите все правильные ответы:
Неверно. Результат одинаков, но метод строк join() делает все копирования в один проход. В такой ситуации рекомендуется всегда использовать его.
Неверно.
