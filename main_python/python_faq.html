Вопросы и ответы по python

  1. Где можно пройти бесплатное тестирование по python?
    1. Сертификация от IT.mail.ru
    2. на сайте upwork
  2. Как получить requirements.txt со всеми зависимостями проекта?
    1. Чтобы получить список пакетов в проекте выполняем команду pip freeze
    2. Для записи вывода в requirements.txt дополняем следующим образом: pip freeze > requirements.txt
  3. Как установить такую-то python библиотеку на Windows, которую не установить через pip?
    1. Есть специальный ресурс http://www.lfd.uci.edu/~gohlke/pythonlibs/ посмотрите может там есть нужный Вам дистрибутив.
  4. Ресурсы, где можно выучить python бесплатно?
    1. Курс на платформе Stepic
    2. Курс от майкрософт на платформе MVA (с русскими субтитрами)
