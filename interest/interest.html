Интересные вещи в сети

  1. Oppia. Обучение с помощью экспертной системы. Написан на питоне. Можно самому составлять программу обучения не имея навыков в программировании. Есть статистика по обучающимся, которая позволяет улучшать систему обучения.
  2. Untrusted. Игра, где интерактивно изменяя javascript код нужно решать головоломки.
  3. Переведем coursera! Перевод онлайн курсов на русский язык.
  4. TED.com. На русском языке.
  5. Hacker News. Хакерские новости.
  6. Игра 2048.
  7. AvaxHome. Обучающие видео курсы.
  8. Портал открытых данных. Города Москвы.
  9. Хаб открытых данных.
  10. Квантовая онлайн песочница от Google. Статья на Хабре.
  11. Разделение визуально и доходчиво. Достойно перевода на русский язык.
  12. Создание сайта проекта или организации на pages.githab.com.
  13. Много материала про поиск пути и других алгоритмах для напиания игр. Много ссылок. Red Blob Games.
  14. Сервис для создания своих опросов и тестов, которые можно встраивать на свой сайт и по которым ведется минимальная статистика Gistroll.com.
  15. Визуализатор регулярных выражений на javascript regulex.
  16. Игры и головоломки на разных платформах
    1. CodeMoo - головоломки для изучающих java.
    2. Code Hunt - головоломки для программистов на java или C#.
    3. CodeCombat - головоломки для программистов на python, javascript и т.п. Есть мультиплеер.
    4. Regex Crossword - кросворд на тему регулярных выражений.
    5. Crossword - кросворд на тему регулярных выражений. Сверстан на CSS3. Статья на хабре
    6. Elevator Saga - головоломки для программистов на javascript.
  17. GGP.org.
    1. Alloy GGP blog
    2. Parametric Player
    3. Tiltyard. Сервер для игр.
    4. Как создать своего игрока.
