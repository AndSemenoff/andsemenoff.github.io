Расчет ГО для фьючерсного контракта

Общие вопросы

ГО(гарантийное обеспечение) для фьючерсного контракта, заключаемого по расчетной цене, находится по формуле:

\begin{equation} ГО = ГО_{\text{в пунктах}} \times \frac {MisStepPrice}{MinStep} \times (1 + \frac{R}{100})\\ \end{equation}
    Где:
  • \( MinStep \) – минимальный шаг цены;
  • \( MinStepPrice \) – стоимость минимального шага цены;
  • \( R \) – радиус валютного курса;
\begin{equation} R = 2 \times \frac {ГО_{Т-1}}{РЦ_{Т-1}} \end{equation}
    Где:
  • \(ГО_{Т-1} \) – базовый размер гарантийного обеспечения, установленный в вечерней клиринговой сессии предыдущего торгового дня Т-1 по основному фьючерсному контракту на курс рубль-доллар;
  • \( РЦ_{Т-1} \) – расчетная цена, установленная для вечерней клиринговой сессии предыдущего торгового дня Т-1 по основному фьючерсному контракту на курс рубль-доллар.
Здесь R – радиус валютного курса (обуславливается риском изменения валютного курса). Алгоритм расчета радиуса валютного курса упоминается на странице.
ГО в пунктах равно 2 * лимит. Здесь (http://moex.com/s206#param) приведены минимальные значения лимитов. Фактические значения лимитов определяются в соответствии с методологией, описанной здесь: http://moex.com/a1463
Готовые данные о лимитах и размерах ГО публикуются на странице гарантийное обеспечение по фьючерсным контрактам.
Для контрактов, заключаемых по цене отличной от расчетной, ГО находится по методологии

Пример расчета

Приведем пример расчета ГО на индекс РТС
В quick транслируется ГО при покупке фьючерса по расчетной цене: ГО = 2 * L * (MinStepPrice/ MinStep) * (1 + R / 100) = 8560 * (12.81 / 100) * 1.16 = 12 721,92 (различия в последних знаках из-за большой точности расчета переменных)
