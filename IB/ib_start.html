Общие вопросы

Скачиваем и устанавливаем последнюю версию Trader Workstation(TWS) API - ссылка

Устанавливаем последнюю версию python

Переходим в директорию(если не меняли по умолчанию):

C:\TWS API\source\pythonclient>

И вводим команду: python3 setup.py sdist

Настройки TWS для работы с API

Включаем "Enable ActiveX and Socket Clients"

По умолчанию TWS использует 7496 порт для реальной торговли и 7497 порт для игровых торгов. IB Gateway по умолчанию использует
4001 порт для реальной торговли и 4002 для игровых торгов. Эти значения можно изменить в пункте "Socket Port" и далее в программе указывать в параметрах функции connect(). Далее ставим галочку в пункте "Allow Connections from Localhost Only". Тем самым мы разрешаем подключение к TWS только с локального компьютера и в этом случае не нужно дополнительного подтверждения входящего подключения на стороне TWS.
